# Skill/Workflow Complexity Definitions

This document provides definitions used to guide review depth in the skill-review workflow.

## Complex Skill/Workflow

- More than 200 lines in main file
- Multiple reference files (3+)
- Multiple domains or technologies addressed
- Requires comprehensive review with all perspectives

## Simple Skill/Workflow

- Fewer than 100 lines in main file
- Single domain or well-understood technology
- Minimal or no reference files
- Can use Quick Review mode

## Intermediate Skill/Workflow

- 100-200 lines in main file
- 1-2 reference files
- Single domain with moderate complexity
- Use judgment for review mode selection
